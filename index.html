How to get your Client ID (free, ~5 min)

1
Go to console.cloud.google.com/apis/credentials and create a project (or use an existing one).
2
Click Enable APIs → search for Gmail API → Enable it.
3
Go to Credentials → Create Credentials → OAuth Client ID → choose Web application.
4
Under Authorised JavaScript origins, add null (for local file use) OR the URL where you're hosting this file.
5
Copy the Client ID and paste it above. Done.
6
Tip: In OAuth consent screen, add your Gmail address as a test user so you can authenticate immediately without Google review.
